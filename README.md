# Lezzet Defteri

Lezzet Defteri, en sevdiğiniz yemek tariflerini listeleyebileceğiniz, favorilerinize ekleyebileceğiniz, yeni tarifler oluşturabileceğiniz ve QR kod tarayıcı ile hızlıca tariflere veya dış bağlantılara erişebileceğiniz modern bir React Native (Expo) mobil uygulamasıdır.

## Özellikler

- **Tarif Listesi:** Uygulamada kayıtlı tüm tarifleri görebilir, filtreleyebilir ve arayabilirsiniz.
- **Tarif Detayı:** Tariflerin hazırlama süresi, zorluk derecesi, malzemeleri ve yapılış adımlarını inceleyebilirsiniz.
- **Favoriler:** Beğendiğiniz tarifleri favori listenize ekleyebilir ve hızlıca erişebilirsiniz.
- **Tarif Ekleme:** Malzemeleri, yapılış adımlarını ve görsellerini girerek kendi tariflerinizi oluşturabilirsiniz.
- **QR Tarayıcı:** Restoran menülerini, web bağlantılarını veya uygulama içi özel yönlendirme QR kodlarını hızlıca tarayabilirsiniz.

## Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz.

### Gereksinimler

- Node.js (v18 veya üzeri önerilir)
- Expo Go uygulaması (Mobil cihazınızda test etmek için)

### Adımlar

1. **Projeyi klonlayın:**
   ```bash
   git clone https://github.com/furkanusakpinar/LezzetDefteri.git
   cd LezzetDefteri
   ```

2. **Bağımlılıkları yükleyin:**
   ```bash
   npm install
   ```

3. **Projeyi başlatın:**
   ```bash
   npx expo start
   ```

4. **Mobil Cihazda Çalıştırma:**
   - Terminalde çıkan QR kodu telefonunuzun kamerası veya **Expo Go** uygulaması ile taratarak projeyi canlı olarak test edebilirsiniz.
